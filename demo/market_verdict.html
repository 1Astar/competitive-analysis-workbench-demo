市场判定

独立模块：真实需求 · 市场阶段 · 蓝海/红海/伪蓝海 · 购买动机拆分。
已并入主工作台「调研报告」生成流程（默认开启 📊 市场判定）；此页可用于单独导入 JSON 查看或重跑。

判定框架（内置）
  • 真实需求：有人痛、有人找、有人买 — 搜索 / 替代方案 / 痛点 / 付费 / 频率
  • 市场阶段：概念期 → 验证期 → 成长期 → 成熟期 → 衰退期
  • 竞争类型：蓝海 / 红海 / 伪蓝海（须回答「为什么没人做」）
  • 购买动机：陪伴 / 儿童 / 老人 / 桌面礼物 / 智控 / 宠物替代
  • 机会公式：需求强度 × 付费意愿 × 人群规模 × 方案不满 ÷ 获客难度
